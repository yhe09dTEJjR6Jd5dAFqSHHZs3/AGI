这是一个通用游戏AI。

仅用1个Python脚本实现所有功能，严禁使用注释或OCR。

出现问题时，在控制台输出错误信息。

初始化阶段：
自动安装缺失库。
检查文件是否存在：若缺失，自动生成。
生成的所有文件，均存放于电脑桌面上的AAA文件夹及其子文件夹下（包括但不限于：经验池、AI模型文件）

一切就绪后，进入学习模式。
学习模式：持续记录以下数据到经验池：时间戳、电脑屏幕画面、鼠标（左键按下时的位置、左键按下时的时间戳、左键释放时的位置、左键释放时的时间戳、轨迹）、操作来源（人）

用户在控制台输入“睡眠”时，如果此时程序正处于学习模式：暂停记录数据，进入睡眠模式。
睡眠模式：基于经验池中的数据（时间戳、电脑屏幕画面、鼠标、操作来源），开始优化AI。
需要确保当经验池数据量极少时，也能优化成功。
优化完成后，保存AI模型以及相关数据，进入学习模式。

用户在控制台输入“训练”时，如果此时程序正处于学习模式：最小化控制台窗口，进入训练模式。

训练模式：
①持续记录以下数据到经验池：时间戳、电脑屏幕画面、鼠标（左键按下时的位置、左键按下时的时间戳、左键释放时的位置、左键释放时的时间戳、轨迹）、操作来源（AI）
②AI在电脑屏幕上输出鼠标操作（点按/长按/拖动）。

用户按下任意键盘按键时，如果此时程序正处于训练模式：暂停记录数据，暂停输出鼠标操作，进入学习模式。

设CPU、MEM、GPU、VRAM占用率中的最大值为M。
建立一个针对截图频率（1Hz ~ 100Hz）、时序记忆窗口长度（1帧 ~ 100帧）的调节机制（1Hz）。
当M＞90%时，逐渐降低上述参数，直到M≤60%。
当M＜10%时，逐渐提高上述参数，直到M≥40%。

我的电脑：
显存上限：4GB
内存上限：16GB
操作系统：Windows 11
显示器分辨率：2560×1600
GPU：NVIDIA GeForce RTX 3050 Ti

AI“看到”的画面分辨率：256×160。
当经验池大小＞20GB时，删除最旧数据，直到经验池大小＜20GB。

需要避免的报错：
FutureWarning: The pynvml package is deprecated. Please install nvidia-ml-py instead.
